Promptify - AI Prompt Refiner

Description:
Promptify helps you rewrite, improve, and polish AI prompts directly in Chrome using Gemini Nano (or mock fallback).

Features:
- Rewrite prompts for clarity and creativity
- Context menu for one-click text refinement
- Works offline (mock mode)
- Clean popup UI
- Automatically copies refined text
